Pagination


                <cns-pagination ng-model="currentPage" total-pages="23"
                                no-first no-prev no-next no-last
                                first-text="«" prev-text="<" next-text=">" last-text="»"
                                bootstrap-class="pagination-sm"></cns-pagination>

Settings

ng-model: Current page number.

total-pages: Total pages.

Optional

no-first Hide First button

no-last Hide Last button

no-prev Hide Previous button

no-next Hide Next button

first-text (Default: &laquo;) Text for First button

last-text (Default: &raquo;) Text for Last button

prev-text (Default: &lt;) Text for Previous button

next-text (Default: &gt;) Text for Next button

bootstrap-class (Default: null) Bootstrap class: pagination-lg, pagination-sm.

Pager


                <cns-pager ng-model="currentPage" total-pages="23"
                           rotate aligned
                           prev-text="Previous" next-text="Next"
                           prev-icon="<" next-icon=">" no-icon></cns-pager>

Settings

ng-model: Current page number.

total-pages: Total number of items.

Optional

aligned Hide First button

rotate Hide Last button

no-icon (Default: false) Hide icon

prev-text (Default: Previous) Text for Previous button

next-text (Default: Next) Text for Next button

prev-icon (Default: \u2190) Label for Previous icon

next-icon (Default: \u2192) Label for Next icon

Rating

Rating: 4.0. My rating: {{rating}}. 

                <cns-rating ng-model="rating" rating="4.0">Rating: 4.0. My rating: {{rating}}.</cns-rating>

Settings

ng-model: User rating.

rating: Current rating.

Optional

max: (Default: 5) Max rating.

icon-valued (Default: glyphicon-star) Glyphicon for valued

icon-unvalued (Default: glyphicon-star-empty) Glyphicon for unvalued
